1.修饰符 ：非访问修饰符 static (静态)
  static String name = "张三"; //静态变量
  static void show(){ // }    //静态方法
2.关于静态变量和静态方法的总结：
  1>一个类的静态方法只能访问静态变量；
  2>一个类的静态方法不能够直接调用非静态方法；
  3>如访问控制权限允许，静态变量和静态方法也可以通过对象来访问，但是不被推荐；
  4>静态方法中不存在当前对象，因而不能使用 this，当然也不能使用 super；
  5>静态方法不能被非静态方法覆盖；
  6>构造方法不允许声明为 static 的；
  7>局部变量不能使用static修饰。  

3.静态变量，静态方法  - > 类
     成员变量，成员方法  - >对象

5.this关键字 
   1>什么是this:表示类的当前对象。
   2>this作用:1.通过this调用当前对象任意属性和方法.2.区分相同变量名
    针对类: 对员变量，成员方法
    针对对象: 属性和方法
    针对变量: 变量

7.变量作用域
  1>类变量也叫全局变量, 通过static修饰符修饰的变量,如果要被不同包下的所有类都能访问，需要用public访问修饰符修饰.
  2>实例变量也叫成员变量,
  3>局部变量 (方法内定义的变量，只能用缺省修饰或者final) 
  4>块变量 if, for.
